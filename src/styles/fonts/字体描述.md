## 项目中可使用以下3种字体 ##

### 1. Open sans ( sans-serif ) ###
字源：Google Font
用途：用于home页及login页面中的标题或副标题（h1-h5）
样式：[300(light), 400(regular), 600(semi-bold), 700(bold)]

### 2. Nunito ( sans-serif ) ###
字源：Google Font
用途：可用于页面字体
样式：[300(light), 400(regular), 600(bold)]

### 3. DIN-Regular ( sans-serif ) ###
字源：独立安装字体
用途：可用于页面字体
样式：[regular, bold]
